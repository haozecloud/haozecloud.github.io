1.2、镜像下载-pull命令

从docker指定的仓库下载镜像到本地
用法:
docker pull 镜像名称

1.3、本地镜像查看-images命令

查看本地存储的镜像

1.4、镜像详细信息-inspect命令

显示镜像的详细导入由save保存出来的压缩文件镜像

用法:
docker load -i 镜像压缩文件名称 [镜像名称或者ID]

1.5、本地镜像删除-rmi命令

删除本地镜像库中的某个镜像

用法:
docker rmi [镜像名称或者ID]

1.6、镜像保存-save命令

保存镜像为压缩文件

用法:
docker save -o 压缩文件名称 [镜像名称或者ID]

1.7、镜像载入-load命令

导入由save保存出来的压缩文件镜像

用法:
docker load -i 镜像压缩文件名称 [镜像名称或者ID]
